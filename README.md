# **Installation** 

```bash
git clone https://github.com/DrasticCoder/wallpaper-changer.git
```


## **Steps**
 - Add your wallpapers to home/user/images/wallpapers/ folder 
 - Open terminal and run this commands :

   ```bash
   chmod +x wallpaper-script.sh
   ./wallpaper-script.sh
   ```
### And Enjoy !
